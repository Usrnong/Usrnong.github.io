vue 父子组件传参 | vue 3.2

发表于2023-02-10|更新于2023-07-14|vue教程

组件传参

  • 父传子：调用组件时，使用v-bind（：）给子组件传入参数。子组件通过defineProps接收
  • 子传父：通过defindEmits函数，定义要发布订阅的函数，返回的值与之前$emit类似。
  • 子组件暴露给父组件内部属性defineExpose， 父组件获取子组件实例通过ref从而获取暴露的属性或者方法

全局批量注册组件

  • 全局组件可以使我们在每次使用的时候不用可以import引入，直接在其他组件内使用即可。在vue使用创建app挂载时，用component方法组件组件，多个时使用for循环component组测组件
import { createApp } from 'vue'
import App from './App.vue'
import './assets/css/reset/index.less'
import Card from './components/Card/index.vue'
 
 
createApp(App).component('Card',Card).mount('#app')
