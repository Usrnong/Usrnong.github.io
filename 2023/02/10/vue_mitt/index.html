vue 兄弟组件传参&mitt | vue 3.2

发表于2023-02-10|更新于2023-07-14|vue教程

兄弟组件传参

方式一：

借助父组件传参。例如一个父组件App调用了两个A、B子组件，此时A传参需要先给父组件，再通过父组件传给B组件。

方式二：

Event BUS: 手写一个发布订阅模式，然后挂载到VUE config全局即可使用。

Vue2 可以使用$emit 传递 $on监听 emit传递过来的事件，利用的就是js设计模式的发布订阅模式。

方式三：

利用Mitt第三方库

  • 安装 npm install mitt -S
  • main.ts 初始化,全局总线，vue 入口文件 main.js 中挂载全局属性
  • 使用方法通过emit派发， on 方法添加事件，off 方法移除，clear 清空所有
    A组件派发（emit） B组件监听（on） 监听所有事件（ on(“*”) ） 移除监听事件（off） 清空所有监听（clear）
